## Getting started

Just follow below steps

### Familiar with Git?

clone this repository and `cd` into `react-coin` folder

```
git clone https://github.com/suwigyarathore/react-coin.git react-coin

cd react-coin
```

### Install dependencies

```
npm install
```

### Start development server

```
npm start
```
